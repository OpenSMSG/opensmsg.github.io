Planned Features

OpenSMSG is an open source project designed to bring a 16 year-old game into a modern era. The main goal is to resurrect the original game for posterity. Other planned goals and improvements to the game are listed below:

Isometric View

Imporoved camera to remove shuddering and make navigation around the game easier.

Multiplayer Login System

public or private server login to save golf courses, collaborate with other players, and share courses.

SDL2

Strong reliable rendering engine built around the cross-platform development library Simple Directmedia Layer (SDL2)

Modern Optimizations

Graphics that resemble the original style but are designed for modern systems without being too taxing.

Community Created

OpenSMSG is open source so anyone and everyone can clone and submit improvements and featured to the game.
